# Contributing to Off The Pace

The canonical contributing guide lives at **[.github/CONTRIBUTING.md](.github/CONTRIBUTING.md)** —
GitHub surfaces it automatically when you open a pull request or issue.

It covers local setup (`make setup`), running the pipeline (`make dbt-dev` / `make dbt-test`),
the dbt / ML / ingestion test suites, the seven-term identity invariant, code style, the pull
request process, and the Code of Conduct.
