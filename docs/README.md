# docs/-Docusaurus Documentation Site

Explanation, tutorials, and auto-generated reference for the Off The Pace project.
Deployed at [off-the-pace.onrender.com](https://off-the-pace.onrender.com).

## Site structure

| Section | Purpose |
|---|---|
| `docs/understand/` | Goal & approach, seven-term identity, methodology, limitations, repo tour |
| `docs/learn-dbt/` | 13-chapter tutorial: staging → reference → intermediate → marts, hands-on |
| `docs/machine-learning/` | 8-page ML narrative: features, training, validation, ONNX export |
| `docs/reference/` | **Auto-generated**-models, macros, schemas, CLI, model card MDX |
| `static/img/` | Site images; `undraw_*` assets are default Docusaurus vendored images |

The `reference/` section is generated by `scripts/build_reference.py` from dbt YAML, Python
docstrings, and `ml/model_card.yml`. **Do not edit these MDX files directly**-edit the
source and regenerate. CI fails if the generated files drift (`git diff --exit-code`).

## Files

| File | What it does |
|---|---|
| `docusaurus.config.ts` | Site config: title, URL, navbar, footer, plugins |
| `sidebars.ts` | Sidebar navigation tree |
| `src/css/custom.css` | Theme overrides |
| `conftest.py` | pytest config for any Python tests in this directory |
| `.vale/` | Vale prose linter rules-style checks on documentation |
| `.cspell.json` | CSpell spellcheck dictionary |

## Local development

```bash
cd docs
yarn               # install deps
yarn start         # dev server at http://localhost:3000
yarn build         # production build to docs/build/
```

## Generated / gitignored

`docs/build/` is the static output-gitignored, regenerated by `yarn build` or CI.
`docs/static/img/undraw_*` and `docusaurus.png` are vendored Docusaurus default assets.
