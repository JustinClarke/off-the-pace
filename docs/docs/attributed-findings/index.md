# Attributed Findings

Each entry here is a case study applying the Off The Pace decomposition to a specific real race. The pipeline covers 168 races across 2018–2024; these findings are selected for narrative clarity and domain interest.

The methodology behind every number is documented in [Seven-Term Identity](/understand/seven-term-identity) and [Methodology](/understand/methodology). The full data source is [`fct_lap_residuals`](/reference/models/fct/fct_lap_residuals).
