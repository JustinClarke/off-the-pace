# `int_dirty_air_tax_component`

> For the conceptual foundation, read [Int Dirty Air Tax Component](/understand/methodology).

Dirty air tax model. Per-lap cost of dirty air following via lagged identification. One row per lap. Uses t-1 dirty-air intensity to avoid reverse causality: current lap's slowdown is attributed to the *prior* lap's position, ensuring causal arrow points from prior position to current performance.
