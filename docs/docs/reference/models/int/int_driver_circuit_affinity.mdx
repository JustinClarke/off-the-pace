# `int_driver_circuit_affinity`

Driver circuit affinity model. Bayesian shrinkage of per-(driver, circuit) residual means toward each driver's global mean (prior_weight = 5 virtual races). Identification: within-driver, between-circuit variation across seasons. Output grain: (driver_id, circuit_key). One row per driver-circuit pair. PK: driver_circuit_id (surrogate hash). Sign convention: negative = faster than driver's own global mean at this circuit. affinity_confidence = n_obs / (n_obs + 5); values below ~0.17 are prior-dominated. Upstream: fct_driver_skill_features, race_to_track seed.
