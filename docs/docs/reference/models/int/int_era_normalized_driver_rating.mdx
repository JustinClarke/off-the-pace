# `int_era_normalized_driver_rating`

Era-normalised driver rating model. Cross-era comparable driver rating anchored on bridge drivers. Two-stage: (1) shrunk per-(driver, season) residual from int_driver_season_ratings; (2) era offset calibrated on drivers with ≥8 races in both pre-2022 and post-2022 eras. The 2022 regulation boundary (ground-effect rules) is the era split. era_adjusted_rating: comparable across 2018–2024. Negative = faster than era-normalised field. low_anchor_sample_flag: TRUE if fewer than 3 bridge drivers found (offset set to 0). Upstream: int_driver_season_ratings, bridge driver identification is internal.
