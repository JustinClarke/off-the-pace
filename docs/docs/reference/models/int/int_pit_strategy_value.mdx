# `int_pit_strategy_value`

Pit strategy model. Pit window opportunity cost per stint. One row per stint. Computes optimal_pit_lap (minimising degradation + pit-loss cost), opportunity_cost_s (seconds lost from overrunning), and strategy_verdict. strategy_verdict: optimal / overran / undercut_forced / early / unknown. Assumes cliff model provides the degradation counterfactual; assumes pit_lane_loss_s from circuit_reference.csv (imputed at 21.0 s when unknown).
