# `int_sector_residual_decomposed`

Sector residual decomposition model. Sector-grain residual decomposition. Grain: (lap_id, sector)   3 rows per lap matching stg_sector_times. Allocates lap-level physics components (fuel, compound, rubber, ambient, constructor, dirty_air_tax) proportionally by sector time share. Residual identity: sector_pace_delta_s = sector_total_explained_s
  + sector_driver_skill_residual_s.
NOTE: constructor allocation is proportional (no power/aero split) pending Simulation #6. NOTE: dirty_air allocation is proportional (no per-sector data) using dirty_air_share_lap.
