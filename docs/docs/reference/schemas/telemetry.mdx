# Telemetry

Bronze-layer event log. One row per sample per driver per lap. Sampled at ~10 Hz; ~2–5 million rows per race. Very wide partition   always filter by season and race_id.

## Overview

| Property | Value |
|---|---|
| File pattern | `bronze/telemetry/season=YYYY/race=<slug>/telemetry.parquet` |
| Example row count | 2024 Bahrain: 23 drivers × 57 laps × 300 samples/lap ≈ 3.9M rows |

:::note

speed_kph, throttle_pct, brake_pct are float32 to save space. Sparse: many null samples for cars that DNF or have sensor dropouts.

:::
