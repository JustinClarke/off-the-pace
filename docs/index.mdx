Off The Pace answers the question that lap time alone cannot: **when a car is off the pace, exactly why?** Every lap is split into seven named, additive components each measured in seconds so you can see precisely how much pace came from fuel mass, tyre degradation, aerodynamic wake, ambient conditions, constructor speed, and the driver's own execution.

## How it works

Off The Pace ingests raw F1 telemetry from 168 races across 2018–2024 and runs every lap through a physics-first decomposition pipeline. The result is a single additive identity that holds to within 0.1 ms on every lap:

```
pace_delta = fuel + compound + rubber + ambient + constructor + dirty_air + driver_skill
```

Each term has physical units (seconds). Positive means slower than the field baseline; negative means faster. The seven terms sum to zero not as an approximation, but as an enforced contract tested in CI on every build.
