# `clean_lap_filter`

Reusable WHERE clause predicate for filtering to "clean" laps suitable for driver skill extraction. A "clean lap" is one where: 1. correction_weight = 1.0 (no manual outlier downweighting) 2. anomaly_class ∉ ('mistake', 'conditions') (excludes crashes, water runoff, etc.) 3. is_rain_lap = FALSE (excludes wet-compound laps in mixed-weather races) This macro is the canonical definition. Use it in every model that extracts or depends on driver skill signals to prevent silent divergence where one model accidentally includes rain laps while another filters them out. Used as: WHERE &#123;&#123; clean_lap_filter() &#125;&#125; in a FROM clause.
