# `int_era_normalized_driver_rating`

One row per (driver_id, season). Cross-era driver rating: compares drivers across 2018–2024 despite regulation changes (2022 ground-effect boundary). Uses bridge drivers (≥8 races both eras) to estimate era offset, then adjusts season residuals. era_adjusted_rating = comparable across eras; negative = faster than era average. low_anchor_sample_flag = few bridge drivers (weak offset).
